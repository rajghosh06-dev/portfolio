# 🌐 Rishit Ghosh – Personal Portfolio

This is my modular, dynamic personal portfolio built with HTML, CSS, and JavaScript. It showcases my skills, education, projects, certifications, workshops, and experience—all loaded dynamically from JSON files for easy updates.

---

## 📁 File Structure

```
portfolio/
├── index.html              # Optional landing page
├── home.html               # Main portfolio page
├── about.html              # Personal bio and background
├── contact.html            # Contact details and external links
├── css/
│   └── style.css           # Global styles
├── js/
│   └── main.js             # Dynamic content loader
└── data/
    ├── projects.json       # Modular project entries
    ├── certificates.json   # Certifications & workshops
    └── experience.json     # Academic and technical experience
```

---

## ✨ Features

- Modular content loading via JSON
- Split view for Certifications & Workshops
- Floating “Contact Me” button with scroll-aware visibility
- Inline CTA in Education section
- Responsive layout and semantic HTML
